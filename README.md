
Cryptobalance is a chrome extension for showing the balance of crypto exchanges and wallets.

The supported crypto exchanges are
  - bitfinex
  - binance
  - cex  
  - bithumb
  - korbit
  - coinone

The supported crypto wallets are
  - ethereum(by etherscan)

```
$ npm run build
```
